GENERAL OBJECTIVE

INNOVATED PARKING LOTS

•Create an orderly system for a parking lot which efficiently time to search for a place using indicators for drivers and speed up the payment system for it.• Decrease the time to search for a place. •Provide greater safety to the vehicle. •Reduce pollution by speeding up search times. •Paper reduction

The increase in vehicle acquisition in Mexico, as well as, in other countries has led to a radical and significant increase in vehicle problems, many years ago it has wanted to solve the daily chaos that these cause, this project focuses on vehicle flow control inside a parking lot and solving the loss of time.
